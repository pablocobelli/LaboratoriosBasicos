Bienvenidos a la Página GitHub del proyecto Laboratorios Básicos.

Esta es la página de descarga de los archivos del Proyecto Laboratorios Básicos, que reúne mis apuntes para un curso de Laboratorio en el cuál las herramientas de análisis de datos son construidas por los estudiantes empleando Python.

Recomendaciones para el uso de los archivos de proyecto

En su mayoría, los archivos de este proyecto son Notebooks de IPython. Para el uso (local) de las notebooks y del código implementado o sugerido en ellas, se emplear Python 3.4 ó superior. En particular, se recomienda instalar la distribución Anaconda de Continuum Analytics.

Los principales módulos de Python empleados durante el curso son:

  • numpy,
  • scipy,
  • sympy,
  • matplotlib, y
  • pandas

Qué archivos contiene este proyecto?

Por el momento este proyecto contiene IPython notebooks (y su material asociado) en las siguientes temáticas:

  • Apunte acerca del uso de Cifras Significativas
  • Introducción al uso de Python (para el trabajo en laboratorio)
  • Cómo leer archivos de MotionDAQ empleando Python
  • Cómo procesar lecturas de Photogates para determinar la velocidad de un móvil
