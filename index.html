Bienvenidos a la Página GitHub del proyecto Laboratorios Básicos.

Esta es la página de descarga de los archivos del Proyecto Laboratorios Básicos, que reune mis apuntes para un curso de Laboratorio en el cuál las herramientas de análisis de datos son construidas por los estudiantes empleando Python.

Recomendaciones para emplear los archivos de este proyecto

Se recomienda emplear Python 3.4 o superior, en particular el curso sugiere instalar la distribución Anaconda de Continuum Analitics. 

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
